# Deploy Vercel

## Dominio

`approssodisera.lucidimezzo.it`

## Passaggi

1. Creare progetto Supabase.
2. Applicare `supabase/schema.sql`.
3. Applicare `supabase/policies.sql`.
4. Collegare GitHub a Vercel.
5. Importare il repository `giordmando/rosso-di-sera-staff-app`.
6. Configurare variabili ambiente.
7. Collegare il dominio.

## Variabili ambiente

Vedi `.env.example`.
